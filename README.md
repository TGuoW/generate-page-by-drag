# generate-page-by-drag

通过拖曳生成页面，以及相应的vue代码，使用的ui库为ElementUI

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
